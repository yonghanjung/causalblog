# Structural causal model (SCM) 으로의 여행

이 포스트에서는 Judea Pearl 의 structural causal model 에 대해서 다루고자 합니다.

목차는 다음과 같습니다. 

1. 상관관계와 인과관계 
2. 인과관계의 수학적 정의 
3. 무작위실험: 어떻게 인과관계를 이끌어낼 수 있을까?
